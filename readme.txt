== Installation ==

How to install the plugin and get it working:


Method-A:

1. Go to your wordpress admin "yoursite.com/wp-admin"

2. Login and then access "yoursite.com/wp-admin/plugin-install.php?tab=upload

3. Upload and activate this plugin

4. Now go to admin menu -> settings -> WP SpeedUp

5- Choose CSS or JS

6- Make changes, save and forget about the speed issue.

	
Method-B:

1.	Download the WP SpeedUp installation package and extract the files on

	your computer. 
2.	Create a new directory named `WP SpeedUp` in the `wp-content/plugins`

	directory of your WordPress installation. Use an FTP or SFTP client to

	upload the contents of your WP SpeedUp archive to the new directory

	that you just created on your web host.
3.	Log in to the WordPress Dashboard and activate the WP SpeedUp plugin.
4.	Once the plugin is activated, a new **WP SpeedUp** sub-menu will appear in your Wordpress admin -> settings menu.

[WP SpeedUp Quick Start]: http://www.websitedesignwebsitedevelopment.com/wordpress/plugins/wp-speedup
